
## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key (is optional beacuse i have ai in my code editor if you want to use ai sidebar put api key if not then you can use without it just install dependencies and run the app )
3. Run the app:
   `npm run dev`
